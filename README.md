# 자동차 경주

* 요구조건
    - 주어진 횟수 동안 n대의 자동차는 전진 또는 멈출 수 있다.
    - 사용자는 몇 대의 자동차로 몇 번의 이동을 할 것인지를 입력할 수 있어야 한다.
    - 전진하는 조건은 0에서 9 사이에서 random 값을 구한 후 random 값이 4이상일 경우이다.
    - 자동차의 상태를 화면에 출력한다. 어느 시점에 출력할 것인지에 대한 제약은 없다.

* 기능구현
    - 자동차 position을 업데이트 하는 기능 구현
        - 랜덤 0~9까지 추출하고 4보다 작을 경우 정지(0), 이상일 경우(1)을 반환하는 메소드 구현
        - 처음 자동차 포지션을 1로 초기화 하기위한 init 메소드 구현
        - 한 단계신 진행 되는 move 메소드 구현
    - 변경된 자동차 position에 따른 출력 기능 구현
        - 자동차 포지션에 따른 상태표시(-) 출력 메소드 구현
        - 레이싱에 참여하는 전체 자동차에 대한 상태표시(-) 출력 메소드 구현(위 구현 상용)
    - 레이싱 게임을 진행하는 main 메소드 구현
        - 사용자 입력을 받는 기능 구현
        - 입력한 수 만큼 게임을 진행 되는 반복 기능 구현
